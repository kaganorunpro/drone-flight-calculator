# Drone Flight Time Calculator

A simple Python project that estimates drone flight time based on:

- Battery Capacity (mAh)
- Number of Motors
- Current Consumption per Motor (A)

## Example

Input:
- Battery: 5000 mAh
- Motors: 4
- Current per Motor: 12 A

Output:
- Estimated Flight Time: 6.2 minutes

## Technologies
- Python
